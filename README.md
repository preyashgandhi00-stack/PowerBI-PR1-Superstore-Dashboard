# Project Overview

This project is created using Microsoft Power BI Desktop based on the Sample Superstore Sales Dataset.

The objective of this project is to clean, transform, analyze, and visualize sales data using Power BI while creating an interactive dashboard.

---

# Dataset Information

- Dataset Name: Sample Superstore Sales Dataset
- Source: Kaggle
- Format: CSV
- Total Records: 9,994+
- Tool Used: Microsoft Power BI Desktop

---

# Data Transformations Performed

The following transformations were completed using Power Query Editor:

- Renamed multiple columns
- Verified data types
- Checked null values
- Filtered Segment to Consumer and Corporate
- Split Order ID into multiple columns
- Removed unnecessary columns
  - Row ID
  - Country
  - Postal Code
- Applied cleaned data to the model

# Dashboard Features

## KPI Cards

- Total Sales
- Total Profit
- Total Quantity
- Average Discount

---

## Charts

### Sales by Category
Displays total sales for each product category.

### Profit by Region
Displays profit earned in each region.

---

## Slicers

- Region
- Category

These slicers allow users to interactively filter dashboard visuals.

---

## Filters

### Page Level Filter

Ship Mode includes only:

- Standard Class
- Second Class

### Visual Level Filter

Top 5 Sub Categories by Sales
